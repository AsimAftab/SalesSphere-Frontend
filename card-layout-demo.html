How The Fix Works

The Problem:

Site cards have a circular image (h-20 w-20 mb-4) that takes up vertical space. Party cards without images are shorter, making the grid look uneven.

The Solution:

Add an invisible placeholder div to Party cards with the exact same dimensions as the image:

<div class="h-20 w-20 mb-4 invisible"></div>

Key Classes Explained:

  • h-20 w-20: Matches the image dimensions (80px Ã— 80px)
  • mb-4: Matches the margin-bottom of the image (16px)
  • invisible: Makes the div invisible but keeps it in the layout flow
  • flex flex-col items-center text-center: Centers all content horizontally
  • h-full: Ensures cards stretch to fill grid cell height

Result:

All cards now have the same height, and text on both card types starts at the same vertical position, creating a professional, balanced grid layout across all screen sizes.
